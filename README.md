## Descripción

**SERVIHOGAR** es una aplicación web orientada a la presentación y gestión de servicios del hogar, con una interfaz moderna, responsive y enfocada en la experiencia de usuario.

El proyecto fue exportado desde Figma Maker y adaptado para funcionar como una aplicación web real con flujo profesional: instalación de dependencias, ejecución local, build de producción y despliegue.

---

## Cómo funciona

La aplicación funciona principalmente del lado del cliente (navegador). No requiere base de datos real ni servidor dedicado para operar en modo demo.

### Estado y datos temporales (memoria del navegador)

Para simular el comportamiento de una app real, la aplicación utiliza el **estado en memoria (RAM del navegador)** y/o almacenamiento local del navegador para manejar información durante la sesión, por ejemplo:

- creación y modificación de perfiles temporales
- cambios en información visible del usuario (nombre, datos, foto, etc.)
- envío/recepción de mensajes simulados o notificaciones
- aceptación, cancelación o eliminación de acciones dentro de la app
- navegación entre secciones sin recargar toda la página
